NEXO LOTERIAS

Política de Privacidade

Última atualização: março de 2026

O NEXO LOTERIAS é um aplicativo de apoio estatístico e organização de apostas em loterias. Esta Política de Privacidade descreve como coletamos, usamos e protegemos as informações dos nossos usuários.

1. Informações que coletamos

Coletamos apenas as informações necessárias para o funcionamento do aplicativo:

  • E-mail e senha — para criação e acesso à conta (via Firebase Authentication).
  • Apostas salvas — números escolhidos, modalidade, data e resultado (armazenados no Firebase Firestore).
  • Token FCM — identificador anônimo do dispositivo para envio de notificações push.

2. Como usamos as informações

  • Autenticar o usuário e manter a sessão ativa.
  • Salvar e exibir o histórico de apostas do usuário.
  • Enviar notificações sobre novos resultados de sorteios e prêmios acumulados.
  • Calcular estatísticas e probabilidades com base no histórico de concursos.

3. Compartilhamento de dados

Não vendemos, alugamos nem compartilhamos dados pessoais com terceiros, exceto:

  • Google Firebase — plataforma de backend utilizada para autenticação, banco de dados e notificações. Sujeita à Política de Privacidade do Google.

4. Segurança dos dados

Os dados são armazenados com criptografia em servidores do Google Cloud. Senhas nunca são armazenadas em texto simples — são gerenciadas pelo Firebase Authentication. As regras de segurança do Firestore impedem que um usuário acesse dados de outro.

5. Retenção de dados

Os dados são mantidos enquanto o usuário possuir uma conta ativa. Ao excluir a conta, todos os dados pessoais são removidos permanentemente.

6. Direitos do usuário

O usuário tem direito a:

  • Acessar os dados armazenados sobre si.
  • Corrigir informações incorretas.
  • Solicitar a exclusão da conta e de todos os dados.
  • Cancelar o recebimento de notificações push a qualquer momento nas configurações do dispositivo.

7. Crianças

O NEXO LOTERIAS não é direcionado a menores de 18 anos. Não coletamos intencionalmente dados de crianças.

8. Aviso legal

O NEXO LOTERIAS é um aplicativo de apoio estatístico e organização de apostas. Ele não garante prêmios nem altera a natureza aleatória dos sorteios. O uso do aplicativo é de inteira responsabilidade do usuário.
